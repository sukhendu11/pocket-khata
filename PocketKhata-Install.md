# Pocket Khata 💰

**পকেট খাতা** — A neo-morphic personal finance manager. Track expenses, manage accounts, set budgets, and stay on top of your finances. **All data stays on your device.**

---

## 📥 Installation

1. **Download** the `PocketKhata.apk` file to your Android phone
2. **Open the APK** — your phone will prompt you to install
3. If asked, **enable "Install from unknown sources"** (Settings → Security → Install unknown apps → File Manager / Chrome → Allow)
4. **Tap Install** and then **Open**

No account, login, or internet connection required.

---

## ✨ Features

- **Dashboard** — Net worth, income/expense summary, recent transactions
- **Transaction Management** — Add, edit, and delete income/expense/transfers
- **Account Management** — Bank, Cash, bKash, Nagad, and custom accounts
- **Category Manager** — Pre-configured + custom categories with colors
- **Budget Planner** — Monthly spending limits with progress tracking
- **Savings Goals** — Set targets and track progress
- **Bill Reminders** — Recurring reminders with due-date tracking
- **Analytics** — Pie charts and income vs. expense breakdowns
- **Calendar View** — Transactions and reminders on a monthly calendar
- **Dark Mode** — Toggle between light and dark themes
- **বাংলা ভাষা** — Full Bangla language support
- **PDF Reports** — Generate income/expense reports
- **Backup & Restore** — Export/import all data as JSON

---

## 🔒 Privacy

- **100% offline** — No internet connection needed
- **No account required** — No sign-up, no login
- **No data tracking** — Zero analytics, no telemetry
- **All data stays on your phone** — Nothing is sent anywhere

---

## ℹ️ First Time Use

1. The app opens to the **Dashboard**
2. Tap the **+** button (center bottom) to add your first transaction
3. Go to **Settings** (gear icon) to:
   - Change language (English / বাংলা)
   - Toggle dark mode
   - Export/import backups

---

## 🔄 Moving Data to a New Phone

1. On your old phone: **Settings → Data Portability → Export Full Database (JSON)**
2. Transfer the `.json` file to your new phone
3. On your new phone: Install Pocket Khata, then **Settings → Data Portability → Import Database (JSON)**
